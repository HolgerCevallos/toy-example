# toy-example

Toy example for Robust PCA and Distributional Shift.

## Structure

- R/ : R functions
- src/ : C++ files (Rcpp)
- toy.example.R : Example script

## Requirements

- R
- Rcpp
- RcppArmadillo

## How to Run

1. Clone the repository:

git clone https://github.com/HolgerCevallos/toy-example.git

2. Open R or RStudio and set the working directory to the project folder.

3. Run the main script:

source("toy.example.R")
